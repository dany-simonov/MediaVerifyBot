# Источник

**Платформа верификации медиаконтента на основе ИИ.**

Источник определяет, создан ли контент человеком или сгенерирован искусственным интеллектом. Поддерживаются фото, аудио, видео и текст. Результат включает вердикт, индекс подлинности и объяснение.

## Статус и стратегия

Мы отказались от Telegram-канала как основного входа для пользователей и сфокусировались на веб-платформе. Проект мигрирован на Appwrite Cloud: клиент, хранение и функция анализа работают через Appwrite SDK и Appwrite Functions.

---

## Что реализовано сейчас

- **Веб-кабинет** с авторизацией, историей проверок и отдельной страницей для глубокой проверки текста
- **Большая проверка текста**: детектор ИИ + фактчек/заимствования с пословной подсветкой
- **Big Check**: пакетный кросс-анализ медиафайлов
- **Explainable AI**: осмысленные объяснения с источниками

---

## Реальные модели и цепочки анализа

### Фото
- **Sightengine (genai)** — основная модель
- **HuggingFace** (fallback): `dima806/deepfake-vs-real-image-detection`

### Аудио
- **Resemble Detect** — основная модель
- **HuggingFace** (fallback): `mo-gg/wav2vec2-large-xlsr-deepfake-detection`

### Видео
- **FFmpeg + Sightengine** — анализ
- **HuggingFace Image** fallback при ошибках/лимитах

### Текст
- **Sapling AI** — детекция AI-текста
- **g4f hybrid fact-check** — фактчек/заимствования
	- Каскад моделей: `gpt-5-nano` → `gpt-oss-120b` → `command-r`

Если основной провайдер недоступен, используется fallback, чтобы сохранить стабильность.

## Архитектура и Appwrite

- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS, Zustand, React Router
- **Appwrite SDK**: авторизация, storage, вызов функций
- **Appwrite Functions**: единая точка анализа
- **Локальный API** (FastAPI) используется для разработки и тестов

Миграция на Appwrite позволила унифицировать деплой, хранение и авторизацию без собственного серверного слоя для продакшена.

## Ограничения и качество

- Анализ вероятностный, возможны ошибки — это вспомогательный инструмент
- Для текста точность выше при длинных фрагментах (рекомендуется 200–2000 символов)
- Видео ограничено по длине и размеру в пайплайне
