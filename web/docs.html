API Документация

REST API для интеграции системы верификации медиа в ваши приложения. Проверяйте фото, видео, аудио и текст на подлинность.

Возможности API

  • Изображения — детекция AI-сгенерированных фото (Sightengine, 94.4% точность)
  • Аудио — детекция синтетической речи и voice cloning (Resemble, 99.5%)
  • Видео — покадровый анализ на deepfake (81%)
  • Текст — детекция ChatGPT и AI-текстов (Sapling, 98%)

Base URL

https://mediaverifybot-api.onrender.com

Аутентификация

Все запросы к API требуют ключа аутентификации, который передаётся в заголовке x-api-secret.

Получение API ключа

  1. Откройте Telegram бот @MediaVerifyBot
  2. Отправьте команду /api_key
  3. Скопируйте полученный ключ
ℹ️
Безопасность

Никогда не публикуйте API ключ в клиентском коде. Храните его только на серверной стороне.

Пример заголовка

HTTP Headers 
x-api-secret: your_api_key_here
Content-Type: multipart/form-data

Быстрый старт

Пример проверки изображения на AI-генерацию за 3 шага:

Python

Python 
import requests

API_URL = "https://mediaverifybot-api.onrender.com/analyze"
API_KEY = "your_api_key_here"

def check_image(file_path):
    with open(file_path, "rb") as f:
        response = requests.post(
            API_URL,
            headers={"x-api-secret": API_KEY},
            files={"file": f},
            data={"user_id": 12345}
        )
    return response.json()

result = check_image("photo.jpg")
print(f"Вердикт: {result['verdict']}")
print(f"Уверенность: {result['confidence'] * 100:.1f}%")

JavaScript (Node.js)

JavaScript 
const fs = require('fs');
const FormData = require('form-data');
const fetch = require('node-fetch');

const API_URL = 'https://mediaverifybot-api.onrender.com/analyze';
const API_KEY = 'your_api_key_here';

async function checkImage(filePath) {
  const form = new FormData();
  form.append('file', fs.createReadStream(filePath));
  form.append('user_id', '12345');

  const response = await fetch(API_URL, {
    method: 'POST',
    headers: { 'x-api-secret': API_KEY },
    body: form
  });

  return response.json();
}

checkImage('photo.jpg').then(console.log);

POST /analyze

Основной эндпоинт для анализа медиафайлов (изображения, аудио, видео).

POST /analyze

Параметры запроса

Параметр Тип Обязательный Описание
file File Да Медиафайл для анализа
user_id Integer Да Уникальный ID пользователя
username String Нет Имя пользователя
text_content String Нет Текст для анализа (альтернатива файлу)

Пример ответа

JSON Response 
{
  "media_type": "image",
  "verdict": "FAKE",
  "confidence": 0.87,
  "model_used": "sightengine",
  "explanation": "Обнаружены признаки AI-генерации изображения.",
  "processing_ms": 1250
}

Возможные вердикты

Вердикт Описание
REAL Контент определён как подлинный
FAKE Обнаружены признаки AI-генерации
UNCERTAIN Недостаточно данных для точного определения

Обработка ошибок

API возвращает стандартные HTTP коды ошибок с JSON телом.

Код Описание Решение
400 Неверный запрос Проверьте формат файла или параметры
403 Доступ запрещён Проверьте API ключ
429 Превышен лимит Дождитесь сброса лимита
500 Ошибка сервера Повторите запрос позже
503 Сервис недоступен AI-модель временно недоступна
