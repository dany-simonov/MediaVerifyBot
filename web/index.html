История версий

Выпущено

v0.1.0 — Альфа-версия

Базовый Telegram-бот, интеграция первых моделей детекции для фото и текста.

Выпущено

v0.2.0 — «Источник»

Полный ребрендинг проекта и запуск функциональной веб-платформы. Внедрён интуитивный «Индекс подлинности», добавлена генерация детализированных PDF-отчётов.

Выпущено

v0.3.0 — Масштабирование

Big Check (пакетный кросс-анализ), Drag&Drop загрузчик, тариф Enterprise (B2B API) и тепловые карты (Explainable AI).

Текущая

v0.4.0 — Глобальное обновление

Внедрение системы авторизации (Telegram + Email), редизайн системы карточек с кастомными SVG-иконками и переработка архитектуры страниц.

Скоро

v0.5.0 — B2B & API

Полноценный релиз личного кабинета, запуск публичного REST API для интеграции с медиа-платформами и релиз расширения для браузера.
