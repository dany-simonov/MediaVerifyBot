История версий

Выпущено

v0.1.0 — Альфа-версия

Базовый Telegram-бот, интеграция первых моделей детекции для фото и текста.

Текущая

v0.2.0 — «Источник»

Полный ребрендинг проекта и запуск функциональной веб-платформы. Внедрён интуитивный «Индекс подлинности», добавлена генерация детализированных PDF-отчётов. Существенно улучшен UX бота и создана единая дизайн-система (Dark Theme + Teal).

Скоро

v0.3.0 — Масштабирование

Big Check (пакетный кросс-анализ), Drag&Drop загрузчик, авторизация через Telegram для личного кабинета, тариф Enterprise (B2B API) и тепловые карты (Explainable AI) для визуализации ИИ-артефактов.
